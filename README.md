# Proyecto 00 - Developer Dashboard

## Descripción

Developer Dashboard es el primer proyecto del Máster Personal de Desarrollo de Software con IA.

Este proyecto servirá como panel personal de desarrollo y evolucionará durante el máster para convertirse en una herramienta real de trabajo.

## Objetivos

- Aprender desarrollo de software desde cero.
- Aplicar buenas prácticas de ingeniería.
- Construir una aplicación real paso a paso.
- Utilizar la inteligencia artificial como asistente de desarrollo.
